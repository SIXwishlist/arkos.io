Genesis uses two different iconfont libraries, allowing to use any combination of them in your plugins or applications.

  1. The standard iconfont library can be viewed here, generated and licensed from IcoMoon for use in Genesis. Cite the class name (structured like gen-classname) as the value in the Python or XML template parameters.
  2. FontAwesome is also included, the list of icons can be viewed here. Cite the class names (structured like fa fa-classname) as the value in the Python or XML template parameters
