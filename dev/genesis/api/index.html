genesis.api.event

Decorator function to register event handlers

      >>> class a(object):
      ...     @event('some/event')
      ...     def test1 (self):
      ...         pass
      ...
      ...     @event('other/event')
      ...     def test2 (self):
      ...         pass
      >>> a._events
      {'other/event': 'test2', 'some/event': 'test1'}

class genesis.api.EventProcessor

A base class for plugins suitable for handling UI Events.
You will need to decorate handler methods with event().

event(event, *params, **kwparams)
Calls a handler method suitable for given event.
      >>> class Test(EventProcessor):
      ...     @event('test')
      ...     def test(self, *p, **kw):
      ...         print(kw)
      ...
      >>> t = Test()
      >>> t.event('test', value='test')
      {'value': 'test'}
match_event(event)
Returns True if class (or any parent class) could handle event.
      >>> class Test(EventProcessor):
      ...     @event('test')
      ...     def test(self):
      ...         pass
      ...
      >>> t = Test()
      >>> t._get_event_handler('test')
      'test'
      >>> t.match_event('test')
      True
      >>> t.match_event('test2')
      False
