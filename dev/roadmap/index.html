The following is a general outline as to what features of arkOS are currently planned, and by when they are expected to be ready, organized by version number.

Genesis

Version 0.7 (Beta) - expected Autumn 2014

New plugins:

  • Pump.io social networking server plugin
  • Diaspora Pod server plugin
  • Music Player
  • RetroShare server plugin
  • Tor Gateway plugin
  • MediaWiki webapp
  • Reevio (Tent blog) plugin (?)
  • Cryptocat Server plugin

Core changes:

  • Move authentication system to LDAP and allow compatible plugins to interface with it
  • Serve Genesis over Websockets for faster, more stable connection
  • Stabilization of Mailserver plugin
  • Tor Hidden Services capability for Webapps
  • Display/manage network bandwidth use
  • Initial integration and availability of arkOS CONNECT services (DelugeDNS and Drydock)
  • Themes!
  • Revisions to Alerts framework (Message Center)

Version 0.8 (Beta) – expected Winter 2014/2015

New plugins:

  • Firefox Accounts (Sync) server
  • (more to come)

Core changes:

  • Internationalization framework
  • IPv6 support and advanced network management tools
  • Domain Registration and DNS management framework
  • External Client API

arkOS Client

Version 0.1 – expected Autumn 2014

  • Connect to arkOS: rapidly scan network for running arkOS servers and seamlessly open Genesis connection to chosen one inside client
  • Install arkOS: Install to all currently supported platforms in both single and split boot/data modes
  • Network Browser: Scan network for running arkOS servers

arkOS CONNECT

DelugeDNS Version 0.1 – expected October 2014

  • Initial version
  • Dynamic DNS provisioning and registration through Genesis clients
  • HTTP/HTTPS Port proxying service (TCP coming later)

Drydock Version 0.1 – expected October 2014

  • Initial version
  • Store backups via LAFS (encrypted) corresponding to CONNECT account

Longterm Goals

  • Automate production of software updates and image releases
  • Integration with possible forthcoming “Indie Web Store”
  • Develop a reliable cross-platform solution for using Full Disk Encryption (FDE) with arkOS
  • Creation of an ephemeral zero-knowledge edition, bundled with certain core tools, for high-risk populations
  • Implementation of a wireless tool suite, for mesh networking and creating mobile nodes for internet access or local content sharing
  • Expansion to x86_64/i686 architecture
  • Partnership or direct sale of custom hardware/storage platforms
